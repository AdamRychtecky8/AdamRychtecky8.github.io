Dialed — Mental Performance Platform for Golfers

Founder · Full-Stack Engineer — designed and built end-to-end using Claude, Copilot, and GPT-4o

Most golfers who want to improve their mental game can’t access real coaching — Dialed brings evidence-based mental performance training to anyone with a phone. Built the full stack solo: PostgreSQL schema with RLS as a hard data boundary, server-first Next.js with React Server Components, timezone-aware daily logic, TypeScript strict mode throughout, and canonical registries enforcing single source of truth across the data model, UI, and analytics layer.

Core product is functional and in active alpha: 5-factor daily check-in feeding a rule-based 8-state headspace classifier, 15+ evidence-based drills with an explicit state machine runner and OpenAI TTS voice guidance, SHA256-hashed server-side audio caching to Supabase Storage, post-round reflection, and streak tracking. Every check-in is an immutable labeled time series record and every round reflection is a before/after pair — the data model was built from the start to support ML-driven personalization. Coach and Caddie AI agents fully designed with specs and stubs — Phase 2.

Ensemble Decision-Making Under Correlation: Humans vs. LLM Groups

Honors thesis research in UCSB’s Vision & Image Understanding Lab. Replicated and extended Juni & Eckstein (2015) across 507,600 trials, comparing human groups and LLM ensembles (GPT-4o, Claude, Gemini) on a 2AFC visual detection task. Groups of 12 outperform individuals by up to 15% — but gains plateau at n≈7, explained by high error correlation in 35–40% of model pairs.

WLC vs. majority voting: +3.3% accuracy · Phase 1 complete · Phase 2 proposed

Modeling Perceptual Decisions: Humans vs. LLMs

PSTAT 231 Final Project — class project using data from ongoing honors thesis research

The task was to predict each agent’s actual decision (“present” or “absent”) — not whether the target was truly there. This distinction matters: agents respond to the sensory evidence they see, so modeling their behavior is a different problem from recovering ground truth. Data came from a 2AFC visual detection task where humans and LLMs (GPT-4o, Claude, Gemini) completed identical trials, allowing direct comparison of how each group integrates the same stimulus information.

Two complementary models were fit on engineered evidence features (strength, noise, left–right difference, cue consistency). A GAM provided an interpretable nonlinear baseline — smooth per-feature effects showing how decision probability changes with evidence strength and trial difficulty. CatBoost was added to capture interactions the additive GAM cannot represent: how noise and evidence magnitude combine, how cue alignment interacts with agent identity, and threshold-like patterns in cue reliability.

The sharpest finding from the SHAP dependence analysis was the noise feature. LLMs showed a steep, monotonic positive relationship — larger angular separation between the two lines reliably predicted a “present” response, as if treating trial clarity as a direct cue. Human SHAP values for noise stayed near zero with high variance, indicating far less systematic use of the same signal. Both groups used evidence strength and cue consistency in similar ways; noise was the clearest behavioral separator.

CatBoost AUC: 0.876 (LLM) · 0.720 (Human) — accuracy: 80.0% (LLM) · 67.7% (Human)
